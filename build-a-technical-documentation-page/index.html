HTML Basics

HTML reprezintă limbajul standard de marcare pentru crearea paginilor web.

Elementele HTML definesc structura conținutului, cum ar fi titluri, paragrafe și liste.

Documentele HTML se bazează pe elemente închise între tag-uri, cum ar fi <div> sau <p>.

HTML5 aduce noi elemente semantice precum <header>, <footer> și <section>.

Utilizarea corectă a tag-urilor îmbunătățește accesibilitatea și SEO.

  • Structura de bază a unei pagini
  • Elemente semantice
  • Formulare și input-uri
<!DOCTYPE html> <html lang="ro"> <head></head> <body></body> <h1>Titlu</h1>
CSS Styling

CSS permite stilizarea elementelor HTML prin culori, fonturi și layout-uri.

Se poate folosi pentru a crea layout-uri responsive și a gestiona poziționarea elementelor.

CSS poate fi inclus inline, intern sau extern.

Selectorii permit stilizarea elementelor specifice.

Pseudo-clasele și pseudo-elementele ajută la stilizare avansată.

  • Selecția elementelor
  • Flexbox și Grid
  • Media Queries pentru responsive
JavaScript și Java

JavaScript este un limbaj de scripting folosit pentru dinamica paginilor web.

Java este un limbaj de programare orientat pe obiecte, folosit pentru aplicații complexe.

Evenimentele și manipularea DOM-ului se fac cu JavaScript.

JavaScript permite interactivitate, validarea formularelor și animații.

Java rulează în JVM și nu este legat direct de browser.

  • Funcții și variabile
  • Event Listeners
  • Clase și obiecte
Responsive Design

Designul responsiv ajustează aspectul site-ului în funcție de dimensiunea ecranului.

Se bazează pe grile fluide, imagini flexibile și unități relative.

Permite o experiență bună pe desktop, tabletă sau mobil.

Flexbox și Grid facilitează layout-uri flexibile.

Mobile-first este o strategie recomandată pentru responsive design.

  • Grile fluide
  • Unități relative (%)
  • Design adaptabil
Media Queries

Media Queries permit aplicarea de stiluri în funcție de caracteristicile dispozitivului.

Se folosesc pentru lățime, înălțime, orientare, rezoluție și multe altele.

Se pot combina condiții folosind operatorul and.

Exemple: schimbarea layout-ului pe mobil sau desktop, ajustarea fonturilor.

Media Queries sunt esențiale în designul responsive modern.

  • min-width și max-width
  • orientation
  • resolution
