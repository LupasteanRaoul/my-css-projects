HTML Basics

Fundamental

HTML (HyperText Markup Language) reprezintă limbajul standard de marcare pentru crearea paginilor web și a aplicațiilor web.

Elementele HTML definesc structura și conținutul documentului, cum ar fi titluri, paragrafe, liste, imagini și link-uri.

Structura de bază HTML5

<!DOCTYPE html>
<html lang="ro">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Titlul paginii</title>
</head>
<body>
    <header>
        <h1>Titlu principal</h1>
    </header>
    <main>
        <p>Conținutul principal</p>
    </main>
    <footer>
        <p>Footer</p>
    </footer>
</body>
</html>

Elemente semantice HTML5

  • <header> - Antetul documentului sau secțiunii
  • <nav> - Secțiune de navigare
  • <main> - Conținutul principal unic al documentului
  • <section> - Secțiune tematică de conținut
  • <article> - Conținut independent, auto-suficient
  • <aside> - Conținut tangențial sau sidebar
  • <footer> - Subsolul documentului sau secțiunii

Sfat: Utilizați elementele semantice pentru a îmbunătăți accesibilitatea și SEO-ul site-ului.

CSS Styling

Design

CSS (Cascading Style Sheets) permite stilizarea elementelor HTML prin culori, fonturi, layout-uri și animații.

Metode de includere CSS

/* 1. CSS extern (recomandat) */
<link rel="stylesheet" href="styles.css">

/* 2. CSS intern */
<style>
    body { font-family: Arial, sans-serif; }
</style>

/* 3. CSS inline */
<h1 style="color: blue;">Titlu</h1>

Selectori CSS

/* Selector de element */
p { color: #333; }

/* Selector de clasă */
.button { background: blue; }

/* Selector de ID */
#header { height: 80px; }

/* Selector de atribut */
input[type="text"] { border: 1px solid #ccc; }

/* Pseudo-clase */
a:hover { color: red; }
li:nth-child(odd) { background: #f5f5f5; }

Flexbox Layout

.container {
    display: flex;
    flex-direction: row;
    justify-content: center;
    align-items: center;
    gap: 20px;
}

.item {
    flex: 1;
    padding: 20px;
}

JavaScript & Java

Programare

JavaScript este un limbaj de scripting folosit pentru dinamica paginilor web, rulând direct în browser.

Java este un limbaj de programare orientat pe obiecte, folosit pentru aplicații complexe și backend, care rulează în JVM (Java Virtual Machine).

JavaScript vs Java

Caracteristică JavaScript Java
Tipizare Dinamică Statică
Execuție În browser În JVM
Paradigmă Multi-paradigmă OOP-first
Compilare Interpretat Compilat

Exemplu JavaScript

// Manipulare DOM
document.getElementById('btn').addEventListener('click', function() {
    const element = document.querySelector('.content');
    element.style.backgroundColor = '#f0f0f0';
    element.textContent = 'Buton apăsat!';
});

// Funcție modernă cu arrow function
const sum = (a, b) => a + b;
console.log(sum(5, 3)); // 8

// Manipulare array-uri
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(num => num * 2);
// doubled = [2, 4, 6, 8]

Responsive Design

UI/UX

Designul responsiv ajustează automat aspectul și layout-ul site-ului în funcție de dimensiunea și orientarea ecranului.

Principii cheie

Grile fluide

Utilizează procente în loc de pixeli pentru dimensiuni

Imagini flexibile

Imagini care se scalează proporțional

Mobile-first

Design pentru dispozitive mobile apoi scalează în sus

Unități relative

Folosește rem și em pentru dimensiuni de font

CSS Grid Example

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
    padding: 20px;
}

.grid-item {
    background: white;
    padding: 20px;
    border-radius: 8px;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

/* Responsive adjustment */
@media (max-width: 600px) {
    .grid-container {
        grid-template-columns: 1fr;
    }
}

Media Queries

CSS

Media Queries permit aplicarea de stiluri CSS în funcție de caracteristicile dispozitivului (lățime, înălțime, orientare, rezoluție).

Puncte de întrerupere comune (breakpoints)

/* Dispozitive mobile (portrait) */
@media (max-width: 575.98px) { /* Stiluri pentru telefoane mici */ }

/* Dispozitive mobile (landscape) */
@media (min-width: 576px) and (max-width: 767.98px) { /* Telefoane */ }

/* Tablete */
@media (min-width: 768px) and (max-width: 991.98px) { /* Tablete */ }

/* Desktop */
@media (min-width: 992px) and (max-width: 1199.98px) { /* Desktop mici */ }

/* Desktop mari */
@media (min-width: 1200px) { /* Desktop mari */ }

/* Exemplu practic */
.container {
    width: 100%;
    padding: 10px;
}

@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
        padding: 20px;
    }
}

@media (min-width: 1200px) {
    .container {
        width: 1140px;
    }
}

Orientation și alte caracteristici

/* Orientare landscape */
@media (orientation: landscape) {
    .header { height: 60px; }
}

/* Orientare portrait */
@media (orientation: portrait) {
    .header { height: 80px; }
}

/* Rezoluție înaltă (retina) */
@media (-webkit-min-device-pixel-ratio: 2), 
       (min-resolution: 192dpi) {
    .logo { background-image: url('logo@2x.png'); }
}

/* Mod întunecat al sistemului */
@media (prefers-color-scheme: dark) {
    body { background: #121212; color: #fff; }
}

/* Reducere mișcare (accessibility) */
@media (prefers-reduced-motion: reduce) {
    * { animation-duration: 0.01ms !important; }
}

Best Practices

Recomandări

Accesibilitate (A11y)

  • Folosește tag-uri semantice HTML5
  • Adaugă texte alternative pentru imagini (alt text)
  • Asigură contrast suficient între text și fundal
  • Permite navigarea cu tastatura
  • Utilizează ARIA labels când este necesar

Performanță

  • Minimizează fișierele CSS și JavaScript
  • Compresează imaginile
  • Folosește lazy loading pentru imagini
  • Implementează caching
  • Redu numărul de request-uri HTTP

SEO

  • Folosește structură corectă de heading-uri (h1-h6)
  • Adaugă meta tag-uri descriptive
  • Creează URL-uri semantice
  • Asigură încărcare rapidă a paginii
  • Creează sitemap.xml și robots.txt

Atenție: Testează întotdeauna site-ul pe diferite dispozitive și browsere pentru a te asigura că funcționează corect pentru toți utilizatorii.
